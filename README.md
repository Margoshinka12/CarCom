На основе представленного ниже примера разработать компонент моделирующий работу реального технического объекта (банкомат, шлагбаум на автопарковке, кофе-машина и т.д.). Компонент должен поддерживать три пользовательских интерфейса.
Реализовать клиенты на С++ и C#. Реализовать взаимодействие через пользовательские интерфейсы и интерфейс IDispatch.
